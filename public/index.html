What features are there?

What are some of the features you can experience in 'My Mentor Mentee'?

Emotional analysis

At the end of each lecture, students make a one-line assessment of the lecture. By evaluating students' feelings about the lecture, they will select popular lectures and mentors.

Online meeting

A video conferencing service that lets you connect anywhere, anytime, with hardware! It is also very efficient when doing team projects.

Screen sharing

How interesting is it to be able to share screens in real time

chatting

Users can communicate with each other. Would not it be nice to do something like share a website address?

File transfer

File transfer between users is possible.Share materials for meetings and classes.

Locking

If you want to have a secret meeting, you can lock it to prevent outsiders from coming in.
